СОБЫТИЯ

Вкладка "События" показывает историю использования оборудования. Дата, ФИО сотрудников, название оборудования, действие с оборудованием

ФИЛИАЛЫ

Вкладка "Филиалы" отображает дочерние подразделения, бэк-офисы организации

ОБОРУДОВАНИЕ

Вкладка "Оборудование" показывает компьютерную технику, которая используется в организации

ДОКУМЕНТЫ

Вкладка "Документы" показывает документацию к оборудованию, инструкции, комментарии

СОТРУДНИКИ

Вкладка "Сотрудники" показывает список сотрудников организации, которые пользуются компьютерным оборудованием

СПЕЦИАЛИСТЫ IT

Вкладка "Специалисты ИТ" показывает список программистов и инженеров организации, которые следят за компьютерным оборудованием

КАБИНЕТЫ

Вкладка "Кабинеты" показывает список кабинетов всех филиалов организации
